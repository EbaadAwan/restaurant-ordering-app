# Restaurant Ordering App

This is a solo project that focuses on important JavaScript concepts like updating the page with JavaScript, handling button clicks, using arrays, and working with user input. The purpose of this mini project was to create a restaurant ordering app where users can add food items to their order, remove items from the cart, and complete a simple checkout process using a fake payment form. The menu items are stored in JavaScript and are displayed dynamically on the page based on user interaction. This project idea came from the Scrimba Fullstack Developer Path, where students complete solo projects after finishing different modules and lessons. Because this project was built in the Scrimba environment, there may be extra whitespace when running the program.

# Getting Started

Install the dependencies and run the project

```bash
npm install
npm start
```
